    Notebook Usage¶

    [2]:

    import xepmts

# Needed to display graphical items
xepmts.notebook()

# needs to be set before display
xepmts.panel_settings.GUI_HEIGHT = 500
xepmts.panel_settings.GUI_WIDTH = 1100
